				$dbupdatelog: Path to a query log.  This file must exist and be writable by the web server.  This is useful for determining usage or compiling list of queries for optimization.			
    •

				string
				$dbquerylog: Path to an explain log.  This file must exist and be writable by the web server.  This will log the explain plan and timing information of queries to the file.
